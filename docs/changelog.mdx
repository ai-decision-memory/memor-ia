## New

- **Create workspaces for chats and docs, with pinned prompts per workspace**
  You can create separate workspaces to group related chats and docs, and save pinned prompts for each workspace. Existing chats and docs are assigned to a default workspace so they stay organized.

- **Export docs as Markdown or PDF**
  Docs can now be downloaded as Markdown for editing or reuse, or as PDF for sharing and distribution.

- **See source citations in answers, docs, and exports**
  Chat answers and generated docs now include clickable source citations pulled dynamically from your connected GitHub and Linear sources via MCP, and those citations are preserved in Markdown and PDF exports.

## Improved

- **Edit AI-generated docs after creation**
  After creating a doc with AI, you can now revise its content and title directly in the app. This makes it easier to refine AI-generated drafts without starting over.

</Update>
